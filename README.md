# Aerospace-Trajectory-Optimizer
Numerical trajectory optimization for atmospheric flight using Python
Design Space: "To ensure the flight profile remained within safe operating limits, I integrated a stall speed constraint ($V_{stall}$) into the visualization. This highlights the practical boundaries of the aircraft’s performance envelope."Sensitivity Analysis: "I performed a sensitivity test on the aircraft mass. This demonstrated that a 10% increase in payload significantly shifts the fuel-burn landscape, underscoring the critical nature of weight management in aerospace systems."Trade-off Assessment: "The optimal cruise point is fundamentally governed by the intersection of induced drag (dominant at low speeds) and parasitic drag (dominant at high speeds). This model illustrates the precise altitude where the atmospheric density penalty is balanced by aerodynamic efficiency."
